新版kali安装GNOME桌面环境

发表于 2020-02-10 更新于 2020-03-13

本人是个linux刚入门的新手，文章可能有误或存在争议，反馈邮箱fsquirt@qq.com，非常感谢支持！

一些没用的P话

没错没错，又是一篇讲kali linux的文章
今天一个朋友来问我kali的安装步骤，我就远程帮他安装kali linux
但是，新版kali在安装盘中省略掉了很多东西，这就需要用户在安装的时候联网从kali官方下载别的组件
但下载速度实在是太慢了，一个多小时还没好，我直接断网重新安装kali linux，一路下一步下一步。结果，没有桌面环境。

淦！在选择安装软件时，我明明勾选了default desktop environment(默认桌面环境)。
而且新版kali也自带了Xfce4,但它不启动。我重度依赖图形化界面，毕竟我是新手。去网上查办法，百度的话都是怎么安装远程桌面的桌面环境。
我试了试使用update-alternatives –config x-session-manager来切换桌面，从回显可以看到有三个Xfce4桌面的选项，可我试了都没用！
那就自己来装个漂亮的GNOME吧！

开始！

这里说下，因为安装时选的是中文，而没有桌面环境话中文都是菱形。所以，注意了，别把指令敲错了。

Step1.设置root的密码
首先登录，输入设好的用户名，回车，再输入设好的密码，回车，登录成功然后sudo passwd root设置root的密码
注意：输入指令后，你需要先输一次你在安装过程中设置的账户（也就是你现在登的账户）的密码，回车，然后再输入你设置的root的密码，回车，再重复一遍
执行su，再输入你置的root的密码

Step2.加源
使用vim /etc/apt/sources.list打开源的配置文件

然后需要进入编辑模式，按i、I、a、A、o、O、r和R中某个键即可。

用上下左右方向键移动编辑的位置，在文末加上:
#aliyun
deb http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
deb-src http://mirrors.aliyun.com/kali kali-rolling main non-free contrib
然后按Esc键，再按冒号键（L键右边的那个）+shift，输入wq，回车。

Step3.开始安装GNOME
敲下面的指令：
apt update

apt-get install gnome-core kali-defaults kali-root-login desktop-base
刚敲指令后，会问你要不要安装，按Y，回车。

然后kali就开始安装GNOME了，记得联网，中途会弹出配置界面，按上下键选择gdm3,回车。
当最后一行出现root@kali: ,也就是让你敲指令的时候，GNOME已经安装好了

如果没有让你选择gdm3就直接结束了，最后一行直接是root@kali:了，那就

apt-get update –fix-missing
apt-get install gnome-core kali-defaults kali-root-login desktop-base



来再把GNOME装一次

Step4.能干嘛？重启呀

重启指令：
reboot
然后，桌面环境就出来了。

觉得GNOME太卡？那就换回Xfce吧！（推荐内存小的使用）

还记得之前说kali自带了Xfce4桌面环境吗？我不知道为什么，在我卸载掉GNOME后，Xfce桌面环境就正常工作了！
手动滑稽，新版kali有点意思。。P话不多说，开始
1.在登录界面按下Ctrl+Alt+F3，进入无图形化界面，以root的身份登录
2.sudo apt-get remove gnome-shell 卸载GNOME，在输完这个指令后一会会黑屏，按Ctrl+Alt+Delete重启。此时你就会发现Xfce启动了！我也不知道为什么！但清理工作还没有结束，继续以root身份登录


3.apt-get autoremove 来卸载不必要的依赖关系

4.apt-get remove gnome* 来卸载GNOME的一切

5.apt-get autoclean 和 apt-get clean再清理下
至此，你已经成功换回Xfce4了，它的内存占用更低，对性能的要求也更低。有可能会中文乱码，我的网站里也有解决办法
