手脱UPX壳的一些方法

发表于 2022-08-05 更新于 2022-08-06

查壳

用PEID查壳

寻找OEP

寻找关键语句 popad

单步跟踪

一步一步手动跟踪，向上的跳转不能让它实现，向下反之
Ollydbg 下一步快捷键 F8

遇到向上的跳转方法：
1.在下一行右键 -> 断点 -> 运行到所选位置(F4)
2.右键 -> 断点 -> 切换(F2) 先下一个断点，然后运行，同样的方法取消断点
如果下一行是call 那就再下一行 popad出栈
popad下很大的跳转jmp ， 会跳到 OEP(真正的程序入口点)

ESP定律法

先执行单步，查看窗口右侧ESP的值，如果只有ESP的值是红色的，在右下角命令输入
dd ESP的值 或者 hr ESP的值 回车
或者 右键ESP后的数据 -> 在数据窗口跟随
数值右键 -> 断点 -> 设置硬件访问断点 -> Dword 或 Word -> 运行
调试 -> 硬件断点(H) -> 删除硬件断点
再次单步 到达OEP

2次内存镜像法

打开内存窗口(Alt + M) 或者点击M按钮
找到程序段第一个 .rsrc 下断点，如果没有 就跳过
右键 -> 设置访问中断 -> 运行(shift + F9)
再次打开内存窗口，在OO4O1OO 下断点，再运行
之后执行单步，就能看见popad下很大的跳转，跳转过去OEP

一步简便法(UPX ASPARK)

直接找出栈命令 popad
入口右键 -> 查找(S) -> 命令(C)
不用勾选整个断块
继续执行单步，直到到达OEP

开始脱壳

OD插件脱壳

找到真正程序入口点后 右键那一行 -> 用OllyDump脱壳调试进程
方式1 ： 在内存镜像中搜索JMP[API] | CALL[API]
方式2 ： 在脱壳文件中搜索DLL & API名称

复制第二行修正后的入口点地址

loadpe脱壳

1.右键 -> 修正镜像大小
2.右键 -> 完整转存

然后使用输入表重建程序 将OD插件脱壳修正后的入口点地址复制到OEP中
